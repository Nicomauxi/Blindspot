# Search First

## When to Activate
- The user explicitly asks to use search-first.
- Producing source-backed research or competitive analysis.
- Task context matches: Research-before-coding workflow. Search for existing tools, libraries, and patterns before writing custom code.

## Core Guidance
- Clarify audience, decision, channel, and source requirements before drafting.
- Separate facts from interpretation. Cite sources for research and keep messaging native to each platform or artifact.
- Preserve a consistent voice while adapting structure and length to the medium.

## Patterns
- Compare vendors with citations.
- Summarize a market with risks and confidence.
- Build a source-backed briefing.

## Checklist
- Activation matched the user task and no narrower skill should take precedence.
- Existing project conventions were inspected before proposing or changing implementation.
- Edge cases and failure modes are covered.
- Verification steps are documented or executed.
